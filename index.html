What is Pondasee?

Pondasee is NOT a css framework, it is only our super basic HTML, CSS(sass) Starter w/ custom mixins, UI Kit & Javascript to build a theme. If you want to use this project as your starting point, you need to know basic knowledge of Compass and Sass.
